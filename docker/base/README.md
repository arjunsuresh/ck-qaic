# Base Docker image

Obtain `qaic-docker-1.0.tar.gz` from Qualcomm and extract it to e.g. $HOME.

```
> $(ck find ck-qaic:docker:base)/build.sh
```

## Parameters

- `DOCKER_DIR=$HOME/qaic-docker-v1.0/`
- `DOCKER_OS=centos7`
- `SDK_DIR=/local/mnt/workspace/sdks/`
- `SDK_VER=1.5.6`
- `PLATFORM_SDK`
- `APPS_SDK`
